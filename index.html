Oque é HTML

HTML, é nada mais que uma linguagem de marcação, onde usamos e abusamos de tags, e atributos do html, para criação de websites e aplicações WEB. html significa HyperText Markup Language, em português... Linguagem de marcação de hypertexto.

Semântica básica de um HTML

o html é composto de uma semântica simples começamos com o <html>, o <head>, e o <body>
o html faz com que definamos um conteúdo de uma página
o head faz com que o navegador identifique e entenda que o conteúdo colocado ali dendo do head, seja exécutado primeiro, mas sem mostrar pro usuário.
já o body faz com que o navegador mostre todo o conteúdo pro usuário, no body é onde criamos nosso site, onde usamos nossas tags.
